## Overview

UnitED Calculator is a Java-based application designed to perform calculations and conversions across a wide range of unit types. Built as part of a 5-person team, the project emphasized collaborative development using Agile/Scrum practices, version control, and object-oriented design.

My primary contributions included developing the full user interface in JavaFX and implementing a significant portion of the backend conversion logic, including support for flexible and extensible unit systems.

**Tech Stack:**
- Java (JavaFX & Swing, Object-Oriented Programming)  
- JUnit for unit testing  
- Git & GitHub  
- Agile/Scrum methodology  

## Showcase

UnitED Calculator provides a flexible and extensible system for performing both standard and custom unit conversions through a clean graphical interface.

**Key Features:**
- Support for standard unit conversions across multiple dimensions  
- User-defined custom units with arbitrary ratios  
- Ability to perform complex and compound unit expressions  
- Full-featured GUI built with JavaFX  
- Calculation history and customizable settings  
- Built-in help section for usability  
- Comprehensive JUnit test coverage for accuracy and edge cases
